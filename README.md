# Metrics with prometheus in a Restify-api
simple demo restify api in nodejs to use prometheus metrics

### technology
- node.js v8.11.2
- npm 5.6.0
- restify
- prom-client


### running with docker
```
make build
make run
```
### Testing
http://localhost:3001/hello/krol
```
"hello krol"
```
